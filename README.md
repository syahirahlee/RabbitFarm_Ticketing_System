# RabbitFarm_Ticketing_System
The ticketing system for Rabbit Fun Land is developed and implemented based on system requirement gathering from a real client as part of project design, implementation and evaluation. <br>

This system is built by using Adobe Dreamweaver CC and XAMPP (phpmyAdmin) for database implementation. <br>
All images are taken at the farm location during interview and requirement gathering session.<br>
All logo and design are styled by own ideas to suit client requirements. 


**1. Customer**
- customer must login to purchase ticket and view ticketing details <br>
- customer can browse pages of website for information

**2. Admin Staff**
- book ticket for customers
- search ticketing record based on customer username
- modify ticketing record (update/delete)
- generate report of all customer booking records
